Introduccion

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado en la creación de sitios web para definir la presentación y el estilo de los documentos HTML (HyperText Markup Language). CSS permite a los desarrolladores web separar el diseño y la presentación de la estructura del contenido de un sitio web, lo que permite una mayor flexibilidad y control sobre la apariencia de la página.


Formas de usarlo y sus diferentes propiedades

En cuanto a las formas de usar CSS estarían las siguientes:
1. CSS en línea: Se utiliza la etiqueta HTML style para definir estilos directamente en la página.
2. CSS embebido: Se utiliza la etiqueta HTML style dentro de la etiqueta head para definir estilos en la misma página.
3. CSS externo: Se utiliza un archivo CSS externo para definir estilos que se aplicarán a varias páginas

Entre sus propiedades estarían las siguientes:
• color: Define el color del texto.
• background-color: Define el color de fondo de un elemento.
• font-size: Define el tamaño de la fuente.
• font-family: Define la fuente de la letra.
• text-align: Define la alineación del texto.
• padding: Define el espacio entre el borde y el contenido de un elemento.
• margin: Define el espacio entre el borde y los elementos adyacentes.
• border: Define el borde de un elemento.
• display: Define cómo se muestra un elemento (por ejemplo, display: none lo oculta).
• float: Define la posición de un elemento en relación a otros elementos.
• position: Define la posición de un elemento en relación al documento o al contenedor padre.


Media Query

Una media query en CSS es una técnica que permite adaptar el estilo de una página web a diferentes dispositivos o tamaños de pantalla.
En otras palabras, las media queries se utilizan para definir estilos específicos para diferentes resoluciones de pantalla, dispositivos y condiciones de visualización.
Una media query en CSS se define utilizando la regla @media, seguida de una o varias condiciones que se deben cumplir para que se apliquen los estilos definidos dentro de la media query.
Las condiciones pueden ser cualquier propiedad de la pantalla, como el ancho de la pantalla, la orientación del dispositivo, la densidad de píxeles, etc.



Media Query

Aquí hay una media query que se aplica a pantallas con un ancho máximo de 600 píxeles: 

                        
                            @media (max-width: 600px) {
                            /* Estilos para pantallas con ancho máximo de 600px */
                            }


Margin y Padding en CSS

En resumen, la diferencia principal entre margin y padding en CSS es que padding se aplica dentro de los bordes de un elemento y margin se aplica fuera de los bordes del elemento.
En otras palabras, padding afecta el espacio entre el contenido del elemento y su borde, mientras que margin afecta el espacio entre el borde del elemento y sus elementos adyacentes.

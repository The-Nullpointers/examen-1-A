Introduccion

JavaScript es un lenguaje de programación interpretado y de alto nivel que se utiliza principalmente para desarrollar aplicaciones web interactivas.
Fue creado originalmente por Brendan Eich en Netscape en 1995 y desde entonces ha evolucionado para convertirse en uno de los lenguajes de programación más populares en el desarrollo web.
JavaScript se ejecuta en el lado del cliente (es decir, en el navegador web del usuario) y se utiliza para realizar diversas tareas, como validar formularios, manipular y actualizar el contenido de una página web en tiempo real, interactuar con el usuario, enviar y recibir datos de un servidor web, etc.


Qué es una funcion en JavaScript y cómo se define?

Una función en JavaScript es un bloque de código que realiza una tarea especifica y se puede reutilizar en diferentes partes del código.
Las funciones en JavaScript se definen con la palabra clave function, seguida del nombre de la función, paréntesis que pueden contener parámetros separados por comas, y el cuerpo de la función encerrado entre llaves "{}". 

                        
function sumar(a, b) { return a + b; }

Por ejemplo, aquí hay una función llamada suma que toma dos parámetros y devuelve la suma de los mismos.


¿Cuál es la diferencia entre un array y un objeto en JavaScript?

Un array es una estructura de datos ordenada que contiene un conjunto de elementos, que pueden ser de cualquier tipo de datos, como números, cadenas, booleanos u otros arrays. Los elementos de un array se almacenan en posiciones numéricas secuenciales, llamadas índices, empezando por el índice cero. Se accede a los elementos de un array mediante su índice correspondiente, utilizando corchetes [ ].

Un objeto, por otro lado, es una estructura de datos no ordenada que contiene un conjunto de pares clave-valor, donde cada clave es una cadena y cada valor puede ser de cualquier tipo de datos. Los pares clave-valor se conocen como propiedades del objeto. Se accede a las propiedades de un objeto utilizando la notación de punto . o la notación de corchetes [ ], pasando el nombre de la propiedad como una cadena entre comillas o una variable que contenga una cadena.

La principal diferencia entre un array y un objeto en JavaScript es que los arrays son estructuras de datos ordenadas que se acceden mediante índices numúricos, mientras que los objetos son estructuras de datos no ordenadas que se acceden mediante claves de texto.
